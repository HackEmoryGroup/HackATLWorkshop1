# HackATLWorkshop1

This code is intended for the HackEmory X HackATL workshops. This code is a great way to get started with web developement. 

There is a skeleton of a website with extensive comments in index.html and newStyles.css that teach newcomers
to programming how to change/create content and target elements for styling. 

A finished product can be found at bit.ly/WSdemo1
The product was designed around creating a Slack promotion site.
